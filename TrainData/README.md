# TrainData

`train-inputs.txt` contains 10 short text samples for one-step and small-batch training bring-up.

`whoami-1000.tsv` format:
- one sample per line
- `prompt<TAB>target`
- used by `scripts/Train.WhoAmI.AndExportDat.fsx --train-data ...`

`whoami-1000-unique.tsv`:
- 1000 unique prompts (no prompt duplicates)
- target fixed to `我是 F# 之神`
- intended for anti-loop/anti-template-overfit retraining baseline

`whoami-1000-natural.tsv`:
- 1000 prompts with mixed style (natural/chat-heavy by default)
- target fixed to `我是 F# 之神`
- generated by `scripts/Generate.WhoAmINaturalData.fsx`
- mix ratio: natural 75%, control-style 15%, multilingual 10%
